NablaCAD – AI Powered 3D CAD System

NablaCAD is an AI-driven 3D modeling system designed to convert natural language into parametric 3D models. Built with a combination of OpenCASCADE, custom geometry logic, and ML-based reasoning.

AI Capabilities

  • Natural language → 3D solids, sketches, features
  • Smart design suggestions and auto-corrections
  • Live preview for each command

Core Features

  • 3D visualization using Qt + OpenGL
  • Parametric command replay (custom NL format)
  • Import/Export: STEP, IGES, STL
  • Supported ops: Extrude, Loft, Chamfer, Fillet, Hole, Offset

NablaDraft – AI Powered 2D Drafting Tool

NablaDraft automates 2D sketching and drafting using natural language instructions. It supports AI-based layout creation, editing, layer management, and geometry transformations.

AI Capabilities

  • Natural Language → 2D sketches, edits, trims, offsets
  • Auto layer/color assignment from single prompt
  • Pattern, hole arrays, feature edits
  • AI-driven Image → DXF converter (MVP in progress)

Core Features

  • DXF Import & Export
  • Smart dimensions & annotations
  • Attribute editing via prompt
